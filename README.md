# mc-moddded-lite-server


## Prerequisites 
  - Docker
  - Docker Compose

## Installation

```sh
wget -O docker-compose.yml https://raw.githubusercontent.com/myrddraall/mc-moddded-lite-server/main/docker-compose.yml
```


## Startup

```sh
docker compose up -d
```

## Shutdown

```bash 
docker compose down
```


## View Logs

```sh
docker compose up -d
```
